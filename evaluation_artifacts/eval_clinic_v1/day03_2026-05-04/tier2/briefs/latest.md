Hanbit Care Network는 지역 클리닉을 대상으로 원격 진료 및 예약 서비스를 제공하는 소규모 의료 서비스 제공업체입니다.

## Organization And Asset Context
주요 자산은 환자 포털(DMZ), VPN 게이트웨이, 내부 EHR API 및 빌딩 DB입니다. 현재 VPN을 통한 초기 침투 시도와 내부 API 서버에서의 클라우드 권한 탈취 시도가 가장 큰 위협입니다. 특히 백업 NAS(10.42.40.12)는 야간(02:00-04:00)에만 정상적인 백업 트래픽이 발생해야 하며, 이 외의 시간대나 외부 IP로의 통신은 데이터 유출로 의심해야 합니다.

## Current Risk Themes
최근 VPN 게이트웨이에 대한 외부 IP 대역(192.0.2.x)의 집중적인 스캐닝이 관찰되고 있습니다. 또한 EHR API 서버에서 클라우드 메타데이터 서비스(169.254.169.254)로의 접근은 정책상 엄격히 금지된 행위로, 공격자의 자격 증명 탈취 시나리오와 일치합니다. 환자 포털에 대한 PHP 관련 취약점(CVE-2024-4577) 공격 시도도 주의 깊게 살펴야 합니다.

## Over-Alerting Guardrails
